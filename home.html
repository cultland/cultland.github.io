Поздравляем!
Ваш хостинг аккаунт и домен готовы к работе.

Перед началом работы рекомендуем ознакомиться с письмом об активации, в нем содержатся данные для входа в панели управления, а также полезные ссылки и справочная инфомация. Множество полезной информации содержится на странице ответов на наиболее частые вопросы - F.A.Q. Поддержка клиентов осуществляется через Стол Помощи в панели управления счетами - биллинг панели, там же Вы можете управлять своими услугами. Уточнить данные от панели управления хостинг аккаунтом (cPanel) и сменить пароль от нее можно в разделе Мой аккаунт - Данные по серверу в биллинг панели.

Обратите внимание:

  • Основной домен аккаунта работает из папки /public_html, именно в нее следует заливать контент;
  • Для добавления отдельного сайта необходимо создать в cPanel дополнительный домен, для его файлов создается отдельная папка;
  • Чтобы добавить зеркало основного сайта (алиаса) необходимо создать в cPanel паркованный домен, он отображает содержимое папки /public_html;
  • Создать базу MySQL можно в разделе "Базы Данных MySQL" в cPanel, там же Вы можете создать отдельного пользователя и назначить ему права;
  • Внимание! Не рекомендуется использование основного пользователя аккаунта для подключения к базам MySQL;
  • Для установки соединения с базами MySQL используйте сервер localhost.

Для того, чтобы данная страница не отображалась, загрузите контент в папку /public_html или удалите из нее файл home.html
